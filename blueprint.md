# AI 동물상 테스트 (Animal Face Test)

## 개요

이 프로젝트는 Teachable Machine에서 학습된 인공지능 모델을 활용하여 사용자의 얼굴이 어떤 동물과 닮았는지 분석해주는 재미있는 테스트 사이트입니다. 사용자는 실시간 웹캠 또는 사진 업로드를 통해 인공지능의 분석 결과를 즉시 확인할 수 있습니다.

## 주요 기능 및 디자인

### 1. 인공지능 분석 (AI Analysis)
*   **Teachable Machine 모델:** Google Teachable Machine으로 학습된 모델(`https://teachablemachine.withgoogle.com/models/oC-44fsN3/`)을 사용하여 강아지상, 고양이상 등을 구분합니다.
*   **실시간 웹캠 분석:** 브라우저의 웹캠 권한을 받아 실시간으로 얼굴을 인식하고 분석 결과를 업데이트합니다.
*   **사진 업로드 분석:** 사용자가 직접 사진을 업로드하여 분석할 수 있는 기능을 제공합니다.

### 2. UI/UX 디자인 (Aesthetics)
*   **현대적인 인터페이스:** 깔끔한 레이아웃, 풍부한 여백, 부드러운 애니메이션을 적용한 세련된 UI.
*   **반응형 레이아웃:** 모바일과 데스크톱 환경 모두에서 최적화된 화면을 제공합니다.
*   **다크/라이트 모드:** 사용자의 취향에 맞는 테마 선택이 가능하며, 부드러운 색상 전환 효과를 제공합니다.
*   **시각 효과:** 카드 디자인에 깊이감 있는 그림자(Drop Shadows)와 유리 질감(Glassmorphism) 효과를 적용했습니다.
*   **결과 게이지:** 각 동물상 확률을 시각적인 게이지 바(Progress Bar) 형태로 표시하여 직관적인 이해를 돕습니다.

### 3. 기술 스택
*   **HTML5 & CSS3:** 시맨틱 태그, 현대적인 CSS 레이아웃 (Grid, Flexbox, @layer).
*   **Vanilla JavaScript (ES6+):** 프레임워크 없는 순수 자바스크립트로 구현된 고성능 로직.
*   **TensorFlow.js & Teachable Machine Image Library:** 브라우저 내 인공지능 모델 실행을 위한 라이브러리.
*   **Web Components:** 재사용 가능한 UI 요소를 위한 커스텀 엘리먼트 활용 (선택 사항).

## 진행 단계

1.  **UI 구조 설계:** 인덱스 페이지에 웹캠 영역, 업로드 영역, 결과 표시 영역 구성.
2.  **스타일링:** `style.css`를 통해 테마별 색상과 애니메이션 효과 적용.
3.  **모델 로딩 및 예측 로직:** `main.js`에서 Teachable Machine 라이브러리를 이용해 모델을 불러오고 예측 함수 구현.
4.  **이미지 처리:** 웹캠 스트림 처리 및 업로드된 파일의 `FileReader` 기반 이미지 렌더링.
5.  **결과 시각화:** 분석된 확률 데이터를 기반으로 동적인 결과 화면 구현.
