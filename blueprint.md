# 제휴 문의 및 커뮤니티 사이트

## 개요

이 프로젝트는 현대적인 웹 기술을 사용하여 제휴 문의를 받고 사용자들이 자유롭게 소통할 수 있는 커뮤니티 공간을 제공합니다. Formspree를 통한 제휴 문의와 Disqus를 통한 댓글 기능을 포함하며, 반응형 디자인과 다크/라이트 모드를 지원합니다.

## 구현된 기능

*   **UI/UX:**
    *   CSS 변수와 `oklch`를 활용한 깔끔하고 현대적인 인터페이스.
    *   상단 테마 토글 버튼 (다크 모드 / 라이트 모드).
    *   모바일과 데스크톱을 모두 지원하는 반응형 레이아웃.
    *   세련된 유리 질감(Glassmorphism)과 미세한 노이즈 텍스처를 적용한 컨테이너.
*   **테마 지원:**
    *   **다크 모드 (기본):** 어두운 배경과 고대비 요소.
    *   **라이트 모드:** 깨끗한 흰색 배경과 부드러운 그림자.
    *   `localStorage`를 이용한 테마 설정 유지.
*   **주요 기능:**
    *   Formspree 기반 제휴 문의 제출 기능 (이름, 이메일, 문의 내용).
    *   Disqus 연동을 통한 사용자 댓글 및 커뮤니티 기능.
*   **시각 효과:**
    *   테마 전환 시 부드러운 애니메이션 효과.
    *   다층 그림자와 `:has()` 선택자를 활용한 인터랙티브 요소.
    *   OKLCH 컬러 스페이스를 사용한 선명하고 일관된 색상 표현.

## 기술 스택

*   **HTML5 & CSS3:** 시맨틱 태그, 컨테이너 쿼리(@container), Cascade Layers(@layer) 사용.
*   **Vanilla JavaScript (ES6+):** 프레임워크 없는 순수 자바스크립트 로직.
*   **Formspree API:** 서버리스 백엔드 폼 처리 (제휴 문의).
*   **Disqus:** 커뮤니티 댓글 서비스 연동.
*   **상태 관리:** LocalStorage를 활용한 사용자 설정 저장.
